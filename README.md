## Project Description

This project analyzes user behavior on an EdTech learning platform to evaluate course engagement, completion rates, and student retention.
The analysis was conducted using Google Sheets and SQL queries. 
The goal was to identify key factors influencing course completion, learning progress, and student retention.

## Tools Used

- Google Sheets
- SQL (DBeaver)
- Pivot Tables
- Data visualization

## Dataset

The dataset contains information about student activity on an online learning platform, including:

- course enrollments
- course progress
- course ratings
- time spent learning
- course completion
- user country
- retention after 30 days

## Key Tasks

- Data exploration and validation
- Exploratory Data Analysis (EDA)
- SQL queries for dataset aggregation
- Analysis of course completion rates
- Analysis of student learning progress
- Retention analysis
- Course rating analysis
- Building an analytical dashboard

## Dashboard Metrics

The dashboard includes key learning platform metrics:

- Total number of students
- Course completion rate
- Average learning progress
- Average hours spent learning
- 30-day retention rate

## Key Insights

- Course completion rate was relatively low compared to total enrollments
- Student retention after 30 days was approximately 67%
- Business category courses showed the highest completion rate
- Student engagement varies depending on the course category

This project demonstrates practical skills in data exploration, SQL querying, KPI analysis, and building analytical dashboards.
